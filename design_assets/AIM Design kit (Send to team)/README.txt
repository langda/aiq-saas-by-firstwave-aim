AIM — CANVA KIT
================
Everything here is ready to bring into Canva.

────────────────────────────────────────
1) EDITABLE DECKS  →  "Editable decks (PPTX)"
────────────────────────────────────────
• AIM Brand Guidelines v2.0 — Editable.pptx   (14 slides)
• AIM Supporting Graphics System v1.0 — Editable.pptx  (6 slides)

These are REBUILT natively — every heading and paragraph is a real text
box and the panels/buttons are real shapes, so after importing you can
click and edit the text, change colours, and move things in Canva.

How to open in Canva:
  Canva home → Create a design → Import file → upload the .pptx
  (or  Projects → Add new → Upload)
Note: gradient fills (cover orb, pills, accent blocks) come in as placed
images you can move or swap; all TEXT is editable.

The two matching PDFs are in "Guideline PDFs" for reference / printing.

────────────────────────────────────────
2) EDITABLE ELEMENTS  →  "Elements - SVG"  (vector, best for Canva)
────────────────────────────────────────
Upload via Canva → Uploads → Upload files. SVGs stay crisp at any size
and you can recolor / retype them inside Canva.

  pill_faster / sharper / stronger / aim / apply  — kinetic word pills (edit the text!)
  slash_gradient / slash_coral                     — motion-slash bars
  para_green / para_purple                         — gradient parallelograms
  ring_gradient                                    — gradient ring
  slash_frame                                      — tilted photo/content frame
  numeral_02                                       — outline index numeral
  swatch_brand / coral / purple / amber            — colour swatches

Brand colours (paste HEX into Canva):
  Navy 1E2D4E · Carbon 0E0E12 · Green 6ECB62 · Blue 39A1D5
  Coral FF5E5B→E0249A · Purple 7C5CFF→5B45C9 · Amber FFB23D→FF6F1C

────────────────────────────────────────

────────────────────────────────────────

  Aura:   aura_orb · aura_glow
  Kit:    all pills, slashes, parallelograms, ring, badge_aim, numeral_02
